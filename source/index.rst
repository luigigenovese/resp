Welcome to ReSParse's documentation!
====================================

This preliminary project has the aim of defining a API for dealing with Sparse Matrix operations
in the most flexible way. We try to use SphinX equipped with the sphinx-fortran package with 
the objective of defining the API by using autodoc tools.
